Content Sync

How does AEM Apps use it?

  • Each time the Content Sync cache is updated the application is built from its content
  • Content handlers are used to render or copy each file based on the configuration nodes
  • On startup, mobile application calls to the Content Sync node with current timestamp
  • If new content is available it is downloaded via a zip file and stored to the device using the file API
  • Content is swapped out and application is launched
