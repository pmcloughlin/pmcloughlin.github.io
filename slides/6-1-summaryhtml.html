Summary

How does it all work together?

  • Developer builds application and mobile components based on Angular JS
  • Content author creates application, updates content and activates content
  • Developer replicates the configuration node of the application out to publish
  • Content author updates the Content Sync cache
  • Content Sync config nodes are looped through and all content is requested using content handlers
  • Application is built into zip file at Content Sync cache
  • Developer downloads content cache and uses PhoneGap to build application
  • Application is distributed, upon startup application calls to Content Sync cache for updated content
  • Author updates content, activates, updates Content Sync cache
  • Infintite loop :)
