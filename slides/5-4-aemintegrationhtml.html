Content Sync

  • Adobe Tool for exporting content from repository
  • Used by Digital Publishing Suite (DPS)
  • Exports content in zip format
  • Content is stored with a timestamp
  • Pass a timestamp to Content Sync node and it returns only new content
  • Content handlers are used to request static content and package it up
  • Provides a GUI for clearing and updating the cache
  • Mobile application use the timestamp to only update parts of the application that have changed.
