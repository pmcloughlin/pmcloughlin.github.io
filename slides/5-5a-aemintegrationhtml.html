Angular JS

How does AEM Apps Use it?

  • Applications are built as single page Angular JS apps
  • Individual components are built using Angular JS
  • Each component contains a .jsp for markup and a javascript file for its controller
  • Angular's two way binding is used to communicate between markup and controller
  • As an application is built the .jsp file is transformed into an angular template via a content handler
  • The controller javascript file is added to the project via normal clientlibs functionality
  • Having each component become a template is genius because it solves a major authoring issue
